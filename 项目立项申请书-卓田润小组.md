## 使用技术点

1. 数据库创建、连接及调用（MySQL + MyBatis）
2. 前后端分离架构（Spring Boot RESTful API + Vue.js SPA）
3. 用户登录认证与角色权限控制（Token + RolesAllowed）
4. 数据分页查询与关键字模糊搜索（PageHelper）
5. JSON 数据构建与解析（Jackson ObjectMapper）
6. 数据可视化图表绘制（ECharts 柱状图、饼图）
7. 高德地图集成与逆地理编码（AMap JS API）
8. 文件上传与静态资源映射（MultipartFile + ResourceHandler）
9. Vuex 状态管理与动态路由渲染
10. 表单验证与组件化开发（Element UI）

## 扩展部分

1. 数据面板功能：使用 ECharts 实现医生职级柱状图和医院科室饼图的数据可视化展示，后端通过 Jackson 构建聚合 JSON 数据
2. 高德地图集成：在销售地点管理中集成高德地图 JS API，实现地图/列表视图切换、地图标记展示、点击地图新增地点、逆地理编码自动获取地址等功能
3. 动态路由权限控制：根据用户角色从后端获取菜单列表，前端动态注册路由，实现不同角色看到不同的功能菜单
