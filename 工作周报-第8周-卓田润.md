## 一、实习内容和收获

本周为项目收尾阶段，主要完成部署测试和项目验收：

**1. 项目编译构建（2小时）**
- 对前端项目执行 `npm run build` 进行生产环境构建
- 对后端项目使用 Maven 进行编译打包，生成可部署的 JAR 文件
- 解决构建过程中的依赖冲突和配置问题
- 优化前端打包配置，使用 compression-webpack-plugin 进行 Gzip 压缩

**2. 项目沙箱部署（2小时）**
- 在华为云 CodeArts 平台配置沙箱部署环境
- 部署后端 Spring Boot 应用，配置数据库连接和端口映射
- 部署前端静态资源，配置 Nginx 反向代理
- 验证部署后的系统功能是否正常

**3. 项目云测（2小时）**
- 使用 CodeArts 云测功能对系统进行功能测试
- 编写测试用例，覆盖各模块的增删改查功能
- 测试用户登录、权限控制、数据面板、地图展示等核心功能
- 记录并修复测试中发现的 Bug

**4. 第三阶段评审与项目验收（2小时）**
- 参加第三阶段评审会议，进行项目最终验收
- 演示系统完整功能：登录、数据面板、各管理模块的增删改查、地图展示等
- 展示项目的技术亮点：ECharts 数据可视化、高德地图集成、动态路由权限控制
- 总结项目开发过程中的经验和教训
- 接收评审专家的意见和建议

**收获：** 本周通过项目的编译构建和部署，学习了软件项目从开发到上线的完整流程。了解了 Maven 打包、Nginx 配置、云平台部署等运维知识。通过项目验收，对整个项目有了全局性的回顾和总结。

## 二、劳动任务和体会

经过 8 周的实习，完成了医药信息管理系统的全栈开发工作。从需求分析、数据库设计到前后端编码实现，再到最终的部署测试和项目验收，完整经历了一个软件项目的生命周期。

**主要收获总结：**
- **后端技术：** 熟练掌握了 Spring Boot + MyBatis 的开发模式，理解了 RESTful API 设计规范、Token 认证、角色权限控制等核心技术
- **前端技术：** 熟练使用 Vue.js + Element UI + Vuex 进行前端开发，掌握了组件化开发、状态管理、动态路由等技术
- **扩展技能：** 学习了 ECharts 数据可视化、高德地图 API 集成、Jackson JSON 解析等实用技能
- **工程能力：** 理解了前后端分离架构的设计思想，掌握了 Git 版本控制、项目构建部署等工程化能力
- **团队协作：** 通过项目分组和阶段评审，锻炼了团队协作和项目管理能力

这次实习让我深刻认识到，软件开发不仅需要扎实的编程能力，还需要良好的工程素养和团队协作精神。未来我将继续深入学习，不断提升自己的技术水平和综合能力。
