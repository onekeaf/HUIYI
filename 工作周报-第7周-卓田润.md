# 西南大学本科学生实习工作周志

## 一、实习内容和收获

本周完成了剩余前端页面开发和项目功能扩充：

**1. 医保政策管理页面设计（6小时）**
- 设计医保政策管理页面，实现政策的增删改查功能
- 创建 medicalPolicyInfoManage.js（API 层）和 Vuex store 模块
- 实现表单验证和数据展示

**2. 医药公司政策管理页面设计（6小时）**
- 设计医药公司政策管理页面，实现政策的增删改查功能
- 创建 companyPolicyInfoManage.js（API 层）和 Vuex store 模块
- 处理政策与公司的关联展示

**3. 医生信息管理页面设计（6小时）**
- 设计医生信息管理页面，包含医生的增删改查和密码重置功能
- 创建 doctorInfoManage.js（API 层）和 Vuex store 模块
- 实现医生级别和诊治类别的下拉选择
- 实现密码确认验证和重置密码功能

**4. 必备材料管理页面设计（6小时）**
- 设计必备材料管理页面，实现材料的增删改查功能
- 创建 materialInfoManage.js（API 层）和 Vuex store 模块
- 根据用户角色控制操作权限（管理员可编辑，医生只读）

**5. 主页面数据面板展示功能实现（2小时）**
- 后端创建 DashboardController 和 DashboardService，使用 Jackson 构建 JSON 数据
- 聚合医师人数、药品种类、合作企业、入驻药店等统计数据
- 获取医生职级分布和诊治类型分布数据
- 前端使用 ECharts 绘制医生职级柱状图和医院科室饼图
- 实现最新医保政策和医药公司政策的表格展示
- 安装 echarts@4.9.0 依赖

**6. 销售地点信息地图展示（2小时）**
- 后端修改 Sale 实体类，添加经纬度（lng/lat）和地址（address）字段
- 修改 SaleMapper.xml，支持经纬度和地址的插入
- 前端集成高德地图 API（@amap/amap-jsapi-loader）
- 实现地图/列表视图切换功能
- 实现地图上标记销售地点、点击新增和修改销售地点功能
- 通过逆地理编码自动获取地址信息

**收获：** 本周是收获最丰富的一周。通过实现数据面板功能，学习了 Jackson JSON 解析库的使用和 ECharts 图表绘制。通过集成高德地图，掌握了第三方地图 API 的接入方式，包括地图初始化、标记点添加、点击事件处理和逆地理编码等功能。

## 二、劳动任务和体会

本周完成了所有前端页面的开发和两个功能扩充模块。数据面板和地图展示功能的实现让我接触到了数据可视化和地图开发领域，这些技能在实际项目中非常实用。通过 ECharts 和高德地图的学习，深刻体会到了前端技术的广度和深度，也激发了我对数据可视化方向的兴趣。
