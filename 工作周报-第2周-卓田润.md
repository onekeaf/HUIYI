## 一、实习内容和收获

本周为项目环境搭建阶段，主要完成了以下工作：

**1. CodeArts 项目工程创建与项目规划（4小时）**
- 在华为云 CodeArts 平台上创建项目工程
- 完成项目规划，制定开发计划和里程碑
- 配置项目的任务看板和迭代计划

**2. 后端项目创建及结构初始化（2小时）**
- 使用 Spring Initializr 创建 Spring Boot 项目
- 配置项目依赖：Spring Web、MyBatis、MySQL Driver、PageHelper、Swagger、Jackson 等
- 初始化项目包结构：controller、service、mapper、domain、model、param、config、util 等
- 配置 application.yml，设置数据库连接、端口号等基础配置

**3. 前端项目创建及结构初始化（2小时）**
- 使用 Vue CLI 创建 Vue.js 前端项目
- 安装项目依赖：Element UI、Vuex、Vue Router、Axios、Lodash 等
- 初始化项目目录结构：views、components、api、store、router、utils、style 等
- 配置 axios 请求拦截器和响应拦截器

**4. CodeArts 代码仓库创建与代码托管（4小时）**
- 在 CodeArts 平台创建代码仓库
- 配置 IDEA 和 VSCode 的 Git 代码托管
- 完成首次代码提交和推送
- 学习 Git 分支管理和协作开发流程

**5. 第一阶段评审（2小时）**
- 参加第一阶段评审会议，展示项目规划和环境搭建成果
- 接收评审意见，完善项目配置

**收获：** 通过本周的环境搭建工作，熟悉了 Spring Boot 和 Vue.js 项目的创建和配置流程，掌握了 CodeArts 平台的使用方法和 Git 代码托管操作，为后续的编码开发做好了充分准备。

## 二、劳动任务和体会

本周的劳动主要集中在开发环境的搭建和配置上，虽然没有编写业务代码，但这些基础工作非常重要。通过亲手搭建前后端项目，对 Spring Boot 和 Vue.js 的项目结构有了更深入的理解，也体会到了工欲善其事必先利其器的道理。
